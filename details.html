У меня есть проект, написанный на Python Flask. Ниже представлен код главной страницы. Он работает хорошо ничего менять не нужно, хочу добавить следующий функционал. Я сделал импровизированную кнопку:
При нажатии на нее не должен осуществляться переход по href="{{ url_for('index') }}", а должно появляться всплывающее окно со списком всех приложений, которые есть в проекте и пользователь через это окно должен увидеть к каким приложениям у него есть доступ и выбрать приложения к которым он дополнительно хочет получить доступ и запросить его по нажатию на кнопку. Чтобы было понятнее расскажу сут проекта. Он из себя представляет веб страницу на подобиее окно операционной системы, в этом окне есть иконки приложений, по сути переходы на другие страницы. У каждого пользователя есть доаступ только к отдельным страницам и чтобы получить доступ к новым доступ нужно запросить. Теперь давай с технической точки зрения. По нажатию на кнопку указанную выше, должно появиться всплывающее окно, в нем в виде кнопок должны быть указаны все имеющиеся приложения, список можно получить из БД селектом: SELECT * FROM webplatform.apps ORDER BY id ASC Селект выдаст 3 столбца: id, app и alias. Нам интересен app. В этом поле содержится имя приложения. Соответственно нужно вывести все имена например на сером фоне черным текстом, а те приложения к которым у польователя есть доступ можно отобразить на черном фоне белым текстам. Получить все приложения к которым у пользователя ест доступ можно селектом: SELECT grants FROM webplatform.users WHERE username = {{ current_user.username }} Этим селектом ты получишь список alias приложений (каждому имени приложения соответствует свое сокращенное имя alias), если пользователю даны права на несколько приложений то алиасы будут указаны через точку с запятой, например: RT;IP;. В самом конце точки с запятой может не быть. Также тут может быть указано ADMIN, это значит что пользователь имеет права ко всем приложениям. Данные для подключения к БД postgresql находятся в корневой папке проекта файл config.ini, блок [POSTGRESQL], параметры host, port, database, user, password.
